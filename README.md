# English Vocabulary Pro v1.0

English Vocabulary Pro 是一個使用 HTML、CSS、JavaScript 建立的國中英文單字學習平台。此版本不使用複雜框架，重點是穩定、可執行、可維護，並保留後續擴充空間。

## 功能介紹

- 首頁儀表板：今日進度、總單字數、收藏數、測驗答對率、連續學習天數。
- 單字卡學習：翻卡動畫、上一張、下一張、隨機練習、SpeechSynthesis 英文朗讀。
- 單字資料：已匯入 1315 個國中基礎英文單字與片語，包含詞性、中文意思、分類、程度、例句與翻譯。
- 搜尋與篩選：支援英文、中文、例句搜尋，並可依分類、詞性、熟悉度與收藏篩選。
- 收藏與熟悉度：使用 localStorage 保存收藏與「不熟、普通、熟悉」狀態。
- 快速測驗：英文單字搭配 4 個中文選項，立即回饋並保存答對率。
- 拼字測驗：依中文意思輸入英文拼字，立即顯示正確結果。
- 聽力測驗：播放英文單字後選擇中文意思，可重播發音。
- 閱讀練習：提供國中程度短文、中文翻譯與閱讀理解四選一。
- 文法學習：提供國中常見文法公式、說明、例句與文法小測驗。
- 單字配對：英文與中文點選配對，完成後標記。
- 學習分析：顯示學習數據、收藏、熟悉度、答對率與連續學習天數。
- 成就系統：內建 8 個成就徽章。
- 學生中心 Demo：我的進度、收藏、錯題、今日任務、成就。
- 教師後台 Demo：班級資料、平均答對率、常錯單字、排行榜。
- PWA：提供 manifest.json，支援手機安裝與 standalone 顯示模式。

## 如何安裝

```bash
npm install
```

## 如何啟動

```bash
npm run dev
```

啟動後開啟：

```text
http://localhost:4173
```

## 如何使用

1. 在「學習」頁籤瀏覽單字卡，點擊卡片可翻面。
2. 使用搜尋框與篩選器快速找到需要複習的單字。
3. 點擊收藏按鈕將單字加入我的收藏。
4. 使用「不熟、普通、熟悉」標記每個單字的熟悉度。
5. 到「測驗、拼字、聽力、配對」頁籤進行不同模式練習。
6. 到「閱讀」閱讀短文並回答理解題。
7. 到「文法」複習文法重點並完成小測驗。
8. 到「分析」查看學習統計與成就徽章。
9. 到「學生中心」查看個人進度、收藏與錯題。
10. 到「教師後台」查看 Demo 班級資料。
