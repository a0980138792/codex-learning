# English Vocabulary Pro v1.0

English Vocabulary Pro 是一個使用 HTML、CSS、JavaScript 建立的國中英文單字學習平台。此版本不使用複雜框架，重點是穩定、可執行、可維護，並保留後續擴充空間。

## 功能介紹

- 首頁儀表板：今日進度、總單字數、收藏數、測驗答對率、連續學習天數。
- 單字卡學習：英文、詞性、分類、中文意思、例句、翻譯、翻面動畫與朗讀功能。
- 單字資料：已匯入 1943 個國中 2000 字 PDF 可辨識英文單字與片語，包含詞性、中文意思、分類、程度、例句與翻譯。
- 搜尋與篩選：支援英文、中文、例句搜尋，並可依分類、詞性、熟悉度與收藏狀態篩選。
- 收藏與熟悉度：使用 localStorage 保存收藏、不熟、普通、熟悉狀態。
- 快速測驗：英文單字搭配四個中文選項，立即顯示答題結果並保存答對率。
- 拼字測驗：依中文意思輸入英文拼字，立即判斷並顯示正確拼法。
- 聽力測驗：播放英文發音後，從四個中文選項中選答案。
- 閱讀練習：提供國中程度短文，中文翻譯預設隱藏，按鍵後才顯示。
- 文法學習：提供國中常見文法公式，中文說明與中文翻譯預設隱藏，按鍵後才顯示。
- 單字配對：左側英文、右側中文，正確配對後標記完成。
- 學習分析：今日學習、累積學習、收藏、熟悉、不熟、答對率與連續學習天數。
- 成就系統：內建 8 個學習成就，達成後顯示徽章。
- 學生中心 Demo：我的進度、收藏、錯題、今日任務與成就。
- 教師後台 Demo：班級資料、平均答對率、最常錯單字與排行榜。
- PWA：提供 manifest.json，可支援手機安裝與 standalone 顯示模式。

## 如何安裝

```bash
npm install
```

## 如何啟動

```bash
npm run dev
```

啟動後開啟：

```text
http://localhost:4173
```

## 如何使用

1. 在首頁查看今日學習進度、總單字數、收藏數與答對率。
2. 使用「學習」頁籤翻閱單字卡，可收藏、標記熟悉度與朗讀英文。
3. 使用搜尋與篩選功能，快速找到想練習的單字。
4. 到「測驗」、「拼字」、「聽力」與「配對」頁籤進行練習。
5. 到「閱讀」與「文法」頁籤練習文章與句型，中文內容可按鍵顯示。
6. 到「分析」查看學習成果與成就徽章。
7. 到「學生中心」查看個人 Demo 資料。
8. 到「教師後台」查看班級 Demo 資料。
